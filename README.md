# My DSA Journey

Hey there! 👋

This repo is my personal space for tackling Data Structures and Algorithms problems. It's not perfect, and it's definitely a work in progress - but that's the whole point.

## What's Inside?

My solutions to various DSA problems from LeetCode, HackerRank, and other platforms. Some days I nail it on the first try, other days I struggle and learn from my mistakes. You'll find both here.

## Why This Repo Exists

Honestly? I kept forgetting solutions after a few days, and it was frustrating. So I decided to document everything. This is as much a learning journal as it is a code repository.

## What You'll Find

- 💻 Solutions in Python (mostly)
- 📝 Comments explaining my thought process
- 🔄 Multiple approaches when I found better solutions
- ⏰ Notes on time and space complexity

## Current Focus

Right now I'm working through arrays, strings, stacks, and queues - building that foundation before moving to more complex topics like dynamic programming and graphs.

## Progress Tracking

I'm aiming to solve at least 5 problems a week and review old ones regularly. Some weeks I do more, some weeks life gets busy. It's about consistency

## How did I do this?

This repo is automatically synced using [**Traverse**](https://chromewebstore.google.com/detail/traverse/nnapafjmoelkehjedfgjchoeelgbiama) - a Chrome extension that makes it super easy to push LeetCode solutions directly to GitHub. Huge shoutout to the Traverse team for making the workflow seamless! 🙌

Feel free to browse, learn, or even suggest better approaches! We're all learning here.
